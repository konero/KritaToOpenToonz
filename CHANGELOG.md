# Changelog

## [0.2.0] - 2026-02-03

### Added
- "Flatten animated groups" option for composite group export
- "Include invisible layers" option
- "Include reference layers" option for grey-labeled layers
- "Include non-animated layers" option for static backgrounds

## [0.1.0] - 2026-01-02

### Added
- Export animated layers as separate PNG sequences
- Preserve keyframe timing in OpenToonz X-Sheet
- Cross-platform support (Windows, macOS, Linux)
